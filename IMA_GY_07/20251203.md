# Predikátumlogika
Predikátum: állítás, ahol egy tulajdonságot rendelünk egy alanyhoz  
**Építőelemek:**  
- univerzum: U = { állatok }
- változó jelek: x, y, ... <- alanyok
- predikátum: P<sub>x</sub>: x piros(x: alany)
- kvantorok: ∀, ∃(minden(univerzális), van olyan(egzisztenciális)) → ∀P<sub>x</sub>, ∃P<sub>x</sub>
- függvényjelek: f(x, y) = x + y; f(x) = x<sup>2</sup>
- műveletjelek: kijelentéslogikai műveletek
- segédjelek: (), vessző
- konstans: individuum konstans

## Elsőrendű nyelv
L(P:F)  
P = predikátum halmaz  
F = függvényjelek halmaz

1. mondat → predikátumlogikai formula  
→ = formalizáció
2. predikátumlogikai kifejezés + építőelemek → mondat  
→ = fordítás
3. predikátumlogikai kifejezés + mondat → építőelemek választása

## Gyakorlás
1. Van fehér ló  
univerzum: U = {állatok}  
változó jel: x  
predikátum: F<sub>x</sub>: x fehér; L<sub>x</sub>: x ló  
kvantorok: ∃  
függvényjelek: -  
<br>
∃x(Lx^Fx)  
tagadás: ∀x(¬L<sub>x</sub> v LF<sub>x</sub>)

2. Minden egyetemi hallgató tud úszni  
univerzum: U = {ember}  
változó jel: x  
predikátum: H<sub>x</sub>: x egyetemi hallgató; U<sub>x</sub>: x tud úszni  
kvantorok: ∀  
függvényjelek: -  
<br>
∀x(Hx → Ux)  

3. Bármely két egyenes a síkban vagy párhuzamosak, vagy metszik egymást  
univerzum: U = { geometriai alakzatok }  
változó jelek: x, y  
predikátum: E<sub>x</sub>: x egyenes; P<sub>xy</sub>: x és y párhuzamosak; Mxy: x és y metszik egymást     
kvantorok: ∀  
függvényjelek: -
<br>
∀x∀y((E<sub>x</sub>^E<sub>y</sub>) → (P<sub>xy</sub> v M<sub>xy</sub>))  

4. ∀x∀y((O<sub>xy</sub>^O<sub>yx</sub>) → E<sub>f(x)f(y)</sub>)  
univerzum: U = { Z }  
változó jelek: x, y  
predikátum: O<sub>xy</sub>: x|y; E<sub>xy</sub>: x = y  
kvantorok: ∀  
függvényjelek: f(x): |x|; n = 1 (individuum konstans)  
<br>
Ha kettő egész szám kölcsönösen oszthatóak egymással, akkor egyenlő az abszolútértékük.

5. ∀x(¬O<sub>f(x) n</sub>)  
Minden valós szám négyzete nagyobb vagy egyenlő 0-nál  
univerzum: U = { R }  
változó jelek: x  
predikátum: O<sub>xy</sub>: x < y  
függvényjelek: f(x) = x<sup>2</sup>  
konstans: n = 0
6. Bármely prímszámnál van nagyobb prímszám  
univerzum: U = { számok }  
változó jelek: x, y  
predikátum: P<sub>x</sub>: x prím; N<sub>xy</sub>: x < y  
kvantorok: ∀, ∃  
függvényjelek: -  
<br>
∀x∃y((P<sub>x</sub>^P<sub>y</sub>) → N<sub>xy</sub>)  
vagy!  
∀x∃y(P<sub>x</sub> → (P<sub>y</sub>^N<sub>xy</sub>))
