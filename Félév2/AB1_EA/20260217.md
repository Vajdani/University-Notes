# Fogalmak 
- Adatbázis: a valós világ egy részhalmazának leírásához használt adatok összefüggő, rendezett halmaza
- Adatok tárolása: merevlemezes mágneses háttértárakon
- Adatbázis-kezelő rendszer(ABKR)/Database management system(DBMS): hardver-szoftver rendszer, amely lehetővé teszi az adatok olvasását vagy módosítását

**Relációs Adatbázis-kezelő rendszer**: 
Az adatbázis-kezelő rendszer többfelhasználós, hálózatos környezetben
működő, adatokhoz való hozzáférést, rendszeres és a felhasználói
folyamatok zavartalan működését biztosító szoftveralkalmazás. Gyakran
ABKR-nek rövidítjük.
- Adatok hosszútávú tárolása
- Adatok lekérdezhetősége (bonyolult, kifinomult kérdések)
- Hálózatos környezetbeni működés (pl. weben keresztül elérem az adatokat)
- Konkurens használat (felhasználók azonos időben is elérik, módosítják az adatokat)

DML - Data Manipulation Language 
DDL - Data Definition Language 
DCL - Data Control Language 
TCL - Transaction Control Language 
Ezekbe a nyelvekbe esnek az SQL utasítások

Kulcsok:
- elsődleges kulcs - a rekord beazonosítására használt
- idegen/másodlagos kulcs - egy másik tábla rekordjainak beazonosítására használt 
- superkulcs - kulcsot tartalmazó attribútum halmaz
- összetett kulcs - több adatból álló kulcs

Relációk attribútumai
- Elsődleges attribútum: része a reláció valamely kulcsának
- Másodlagos attribútum: nem része a reláció semelyik
kulcsának
