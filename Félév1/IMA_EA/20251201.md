# Kombinatorika
## Ismétlés nélküli permutáció
6 játékos, mindegyik kap egy kártyát  
Szeretnének az összes lehetséges beosztásban játszani  
6 * 5 * 4 * 3 * 2 * 1 = 720 beosztás  
1 nap = 1 játék, 720/365 = 1.9... év

**Def:** n elem egy sorrendje egy ismétlés nélküli permutációja  
n! =/= n permutációja  
**Tétel:** n elem ismétlés nélküli permutációinak a száma  
P<sub>n</sub> = n! = n * (n - 1) * (n - 2) * ... * 2 * 1  
**Biz:**  
Teljes indukció:  
1. P<sub>1</sub> = 1
2. n = k-ra igaz P<sub>k</sub> = k!  
Megnézzük n = k+1-re  
P<sub>k + 1</sub> = (k + 1) * P<sub>k</sub> = (k + 1) * k! = (k + 1)!  


## Ismétléses permutáció
**Def:** n elem t féle  
1-es: k<sub>1</sub> db  
2-es: k<sub>2</sub> db  
...  
t-es: k<sub>t</sub> db  
Ezeknek egy sorrendje az n elem egy k<sub>1</sub>, k<sub>2</sub>, ... k<sub>t</sub>-ed osztályú ismétléses permutációja  
**Tétel:** n elem k<sub>1</sub>, k<sub>2</sub>, ..., k<sub>t</sub>-ed osztályú ismétléses permutációinak a száma:  
P<sub>n</sub><sup>k<sub>1</sub>, ..., k<sub>t</sub></sup> = (n!)/(k<sub>1</sub>! * k<sub>2</sub>! * ... * k<sub>t</sub>!)  
**Biz:** Először megjelöljük az egyformákat(hogy különbözzenek). Így n! sorrend van. Az 1-es típusról letöröljük a jelet. Ekkor k<sub>1</sub>!-onként ugyanazt a sorrendet kapjuk.  
A 2-es típusról is letöröljük: k<sub>2</sub>!-onként egybeesnek.  
...
A t-es --||-- k<sub>t</sub>! --||--  

## Ismétléses variáció
A seriff 5 lapot kap.  
Megnézi a színét sorban.  
Hányféle dolgot láthatott?  
4 * 4 * 4 * 4 * 4 = 4<sup>5</sup> = 1024  
**Def:** n elemből képezett k hosszúságú sorozat(ismétlést engedve) az n elem egy k-ad osztályú ismétléses variációja.  
**Tétel:** n elem k-ad osztályú ismétléses variócióinak száma: V<sub>n</sub><sup>k, i</sup> = n<sup>k</sup>  
**Biz:** n * n * ... * n = n<sup>k</sup>  

## Ismétlés nélküli variáció
Az 5 lapon sorban megnézi a "számot".  
Kiderül, hogy mind különbüző.  
Hányféle megfigyelést tehetett?  
12 * 12 * 11 * 10 * 9 * (8 * 7 * ... * 1)/(8 * 7 * ... * 1) = 13!/8! = 154440  
**Def:** n különbüző elemből ismétlés mélkül készített k hosszúságú sorozat az n elem egy k-ad osztályú ismétlés nélküli variációja  
**Tétel:** n elem k-ad osztályú ismétlés nélküli variócióinak száma: V<sub>n</sub><sup>k</sup> = n * (n - 1) * ... * (n - k + 1) = n!/(n - k)!  
**Biz:** n * (n - 1) * ... * (n - k + 1) = n!/(n - k)!  

## Ismétlés nélküli kombináció
Az elsőként felhúzott lapok között 5 darab Bang kártya volt. Senki sem kapott 2-t.  
Hányféleképpen valósulhatott ez meg?  
P M* A B C* D* X Y* Z U* V játékosok  
B<sub>1</sub>, ..., B<sub>5</sub> Bang kártyák  
B<sub>1</sub> → C  
B<sub>2</sub> → U  
B<sub>3</sub> → M  
B<sub>4</sub> → D  
B<sub>5</sub> → Y  
↳ n = 11; k = 5; → ismétlés nélküli variáció → 11!/6!  
Így 11!/(6! * 5!) = 465 (ha a Bang kártyák egyformák)  
**Def:** Egy n elemű halmaz egy k elemű részhalmazát az n elem egy k-ad osztályú ismétlés nélküli kombinációja  
**Tétel:** n elem ismétlés nélküli kombinációinak száma: C<sub>n</sub><sup>k</sup> = (n k) = n!/(k! * (n - k)!)  
**Biz:** Ha < konkrét hieroglifák >, akkor n!/(n - k)! db van. Ha nem figyelünk a sorrendre, akkor k!-onként ugyanazt kapjuk. Így n!/(k! * (n - k)!) db kombináció van.
