# Relációk
**Def:** R∈AxB  
Az R halmazt az (A, B) halmazpárok értelmezett kétváltozós(bináris) relációnak nevezzük.  
Jel: A, B, R  

**Megjegyzés:** Legyen a∈A, b∈B és R∈AxB  
Ha (a, b)∈R -> a és b relációbanvan egymással (jel: aRb)  
Ha (a, b) nem∈R -> a és b nincs relációban egymással (jel: a/R/b)  

**Def:** (A, B, R) reláció  
A: indulási halmaz  
B: érkezési halmaz  
D<sub>r</sub> = {a∈A|létezik b∈B, hogy aRb} - értelmezési tartomány  
R<sub>r</sub> = {b∈B|létezik a∈A, hogy aRb} - értékkészlet  

**Pl:**  
A = {1, 2, 3}  
B = {x, y}  
AxB = {(1, x), (2, x), (3, x), (1, y), (2, y), (3, y)}  
R := {(1, x), (2, x)}  
1Rx, 2Rx, 3/R/x
D<sub>r</sub> = {1, 2}  
R<sub>r</sub> = {x}

**Def:** Az (A, B, R) reláció függvény, ha minden a∈A-hoz létezik pontosan egy b∈B, hogy aRb  
Értelmezési tartomány: D<sub>r</sub> = A  
Értékkészlet: R<sub>f</sub> = {y∈B|létezik x∈A:(x, y)∈R}  
**Megjegyzés:** (x, y) rendezett pár esetén  
x: hely  
y: (helyettesítési) érték

Függvények megadása:  
-képlettel  
-táblázattal  
-grafikonnal  
-hozzárendelési szabály szöveges megadásával

# Függvénytulajdonságok
**Def:** Monotonitás: Az f(x) az [a, b]-n  
- monoton növekvő, ha f(x<sub>1</sub>) <= f(x<sub>2</sub>) minden x<sub>1</sub> < x<sub>2</sub>,
- monoton csökkenő, ha f(x<sub>1</sub>) >= f(x<sub>2</sub>) minden x<sub>1</sub> < x<sub>2</sub>,
- szigorúan monoton növekvő, ha f(x<sub>1</sub>) < f(x<sub>2</sub>) minden x<sub>1</sub> < x<sub>2</sub>,
- szigorúan monoton csökkenő, ha f(x<sub>1</sub>) > f(x<sub>2</sub>) minden x<sub>1</sub> < x<sub>2</sub>,
ahol x<sub>1</sub>, x<sub>2</sub>∈[a, b]

**Def:** Korlátosság:  
- f(x) alulról korlátos, ha minden x∈D<sub>f</sub>-hez létezik k∈R, hogy f(x) >= k
- f(x) felülről korlátos, ha minden x∈D<sub>f</sub>-hez létezik K∈R, hogy f(x) <= K
f(x) korlátos, ha alulról és felülről is korlátos

**Def:** Páros/páratlan pár  
- f(x) páros, ha x∈D<sub>f</sub> <=> -x∈D<sub>f</sub> és f(x) = f(-x) minden x∈D<sub>f</sub> /A grafikon szimmetrikus az y tengelyre./
- f(x) páratlan, ha x∈D<sub>f</sub> <=> -x∈D<sub>f</sub> és f(x) = -f(x) minden x∈D<sub>f</sub> /A grafikon szimmetrikus az origóra./

**Def:** f(x) periodikus, ha létezik p∈R\\{O}, hogy x∈D<sub>f</sub> <=> (x + p)∈D<sub>f</sub> és f(x + p) = f(x) minden x∈D<sub>f</sub> /Egy bizonyos rész ismétlésével előáll a függvény./

**Def:** Konvexitás: Az f(x) az [a, b]-n  
- konvex, ha f(x) <= (f(x<sub>2</sub>) - f(x<sub>1</sub>))/(x<sub>2</sub> - x<sub>1</sub>) * (x - x<sub>1</sub>) + f(x<sub>1</sub>) minden a <= x<sub>1</sub> < x < x<sub>2</sub> <= b /A függvény két pontját összekötő húr a görbe fölött van./
- konkáv, ha f(x) <= (f(x<sub>2</sub>) - f(x<sub>1</sub>))/(x<sub>2</sub> - x<sub>1</sub>) * (x - x<sub>1</sub>) + f(x<sub>1</sub>) minden a <= x<sub>1</sub> < x < x<sub>2</sub> <= b /A függvény két pontját összekötő húr a görbe alatt van./

**Megjegyzés:** Szigorúan konvex/konkáv: ha az egyenlőség nincs megengedve.

**Def:** Szélsőértékek:  
- x<sub>0</sub> az f(x) lokális maximumhelye, ha létezik (a, b)∈D<sub>f</sub>, hogy x<sub>0</sub>∈(a, b) és f(x<sub>0</sub>) >= f(x) minden x∈(a, b)
- x<sub>0</sub> az f(x) lokális minimumhelye, ha létezik (a, b)∈D<sub>f</sub>, hogy x<sub>0</sub>∈(a, b) és f(x<sub>0</sub>) <= f(x) minden x∈(a, b)
- x<sub>0</sub> az f(x) globális maximumhelye, ha f(x<sub>0</sub>) >= f(x) minden x∈D<sub>f</sub>
- x<sub>0</sub> az f(x) globális minimumhelye, ha f(x<sub>0</sub>) <= f(x) minden x∈D<sub>f</sub>

**Def:** Zérushely:  
x<sub>0</sub> az f(x) zérushelye, ha f(x<sub>0</sub>) = 0  
/Ahol a grafikon az x tengelyt metszi./
